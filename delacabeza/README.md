# Nombre del Proyecto
deLaCabeza!

## Descripción del Proyecto
deLaCabeza! es un e-commerce que se dedica a la venta de gorras y beanies

### Instalación
git clone “https://github.com/juanferrero13/react“
npm install
npm run dev

### Tecnologías
ReactJS -
 Node.js -
 Vite -
 React Router DOM -
 Firebase/Firestore -
 SweetAlert2 -
 FontAwesome -
 Chakra UI
